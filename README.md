# 🧠 AI Text & PDF Summarizer

A full-stack web application that summarizes text and PDF documents using AI.

## 🚀 Features
- ✏️ Text summarization
- 📄 PDF summarization
- 📥 Download summary as PDF
- 🎨 Clean UI

## 🏗️ Tech Stack
- React.js (Frontend)
- Node.js + Express (Backend)
- OpenAI API

## ⚙️ Run Locally

### Backend
cd backend

node server.js

### Frontend
cd frontend
npm install
npm start
